# Training project Using NodeJS, Express, EJS, MongoDB

## Project Setup
1. **Add dependencies**
   ```sh
   npm install
2. **Add .env file and add following configuration**
   ```sh
   MONGODB_URI = <your_mongodburi_such_as_mongodb+srv://xxxxxxxx>
   PORT = <your_port>
   SESSION_SECRET = <your_session_secret>
3. **Run Project**
   ```sh
   npm run dev
